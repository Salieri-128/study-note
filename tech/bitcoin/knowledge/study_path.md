🎓 MIT 区块链课程 x 编程实战路线图
🚀 第一阶段：创世与底层 (对应课程：第 1 - 5 节)
课程重点： 货币的历史、比特币原理、区块链技术细节（哈希、签名、UTXO）。

技术目标： 彻底搞懂“区块链”作为一种数据结构是如何运作的。脱离了代码，理论只是空中楼阁。

💻 编程作业：【手搓一条 Python 链】

任务： 不使用任何区块链库，仅用 Python 标准库（如 hashlib, datetime）。

要求：

定义一个 Block 类（包含索引、时间戳、交易数据、前一个区块哈希）。

实现一个 calculate_hash 函数。

实现简单的 PoW (工作量证明)：让哈希值必须以 '0000' 开头才能被写入。

通关标准： 你的程序能打印出一条包含 5 个区块的链，并且验证它们的数据哈希是连续正确的。

⚡ 第二阶段：智能合约入门 (对应课程：第 6 - 8 节)
课程重点： 智能合约、以太坊、图灵完备性、DApps。

技术目标： 从“记账”进化到“编程”。开始学习区块链开发的通用语言 Solidity。

💻 编程作业：【你的第一个链上银行】

任务： 使用 Remix IDE（在线编辑器）写一个 Solidity 合约。

要求：

实现 deposit()：允许任何人往合约里存（伪）钱。

实现 withdraw()：只允许存钱的人把自己的钱取出来（防止别人偷钱）。

实现 getBalance()：查询余额。

通关标准： 在测试网成功部署，并能演示存进去的钱，只有对应的账号能取出来。

💰 第三阶段：代币经济学与发行 (对应课程：第 9 - 14 节)
课程重点： 央行与数字货币、支付系统、ICO（首次代币发行）、代币经济学设计。

技术目标： 掌握 FinTech 的核心——资产发行。理解 ERC-20 标准。

💻 编程作业：【发行你自己的数字货币】

任务： 基于 OpenZeppelin 库，编写并发行一个属于你的 ERC-20 代币（比如叫 "StudentCoin"）。

要求：

设定总供应量（比如 100 万枚）。

实现转账功能（Transfer）。

实现“铸币”和“销毁”权限（模拟央行的货币政策）。

通关标准： 把代币发给你的朋友（或者你的另一个钱包地址），并在 MetaMask 钱包里看到余额显示。

🏦 第四阶段：DeFi 金融机制 (对应课程：第 15 - 18 节)
课程重点： 二级市场、交易所、交易、结算。

技术目标： 理解去中心化金融（DeFi）最核心的算法——AMM（自动做市商）。

💻 编程作业：【迷你交易所 (Mini-DEX)】

任务： 模仿 Uniswap 的核心逻辑。

要求：

不涉及复杂的 UI，只写合约逻辑。

实现 addLiquidity（添加流动性）。

实现核心公式 x * y = k：当用户用代币 A 换代币 B 时，价格会根据池子里的比例自动变化。

通关标准： 演示当池子里的币变少时，价格自动变贵。

🌐 第五阶段：全栈与未来 (对应课程：第 19 - 24 节)
课程重点： 监管政策、应用落地、扩展性、总结。

技术目标： 能够开发一个完整的前端+后端（DApp），独当一面。

💻 编程作业：【Web3 众筹平台】

任务： 整合前几个阶段的知识，做一个有网页界面的应用。

要求：

前端： 一个简单的网页，有一个“捐款”按钮。

交互： 点击按钮，自动唤起小狐狸钱包（MetaMask）支付 ETH。

展示： 网页上实时显示众筹进度条。

通关标准： 这是一个完整的 Portfolio（作品集）项目，你可以把它放在你的 GitHub 上求职用。
