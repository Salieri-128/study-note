# 1：关于Fiat Currency历史产生的一些思考和内容

1.1. 我们日常使用的货币，比如说美金，其实是美联储上的总分类账上的负债liability。

1.2. 最初的时候人们会使用美金，一种学说是因为政府说只接受美金用来交税，然后大家就用自己的货物或者黄金什么的，去交换美金，然后美金就有了其内在价值，这个时候政府是获得了这些货物或者黄金的，也就是金本位时期的美金，美联储的纸币意味着总负债，他必须有相同价值的黄金在手。

1.3. 在金本位早期，黄金资产和纸币价值应该对等的，一张定额纸币可以换取一定数量的黄金，但是随着另一种东西出现，黄金资产就不再需要是完全一对一和纸币价值等价，这种东西就是国债

1.4. 国债的出现，其实是“现代国家信用体系”的起点，而法定货币是这个体系的延伸产物。国债诞生政府于向公众借钱（发行债券），以未来税收偿还17世纪起。**政府欠银行的钱成为“国债资产”，银行则发行“纸币”作为负债**

1.5. 在金本位时代：纸币表面上由黄金支撑，但中央银行持有大量国债来调节流动性、赚取利息，所以国债逐渐成为“次级储备资产”。金本位崩溃后国债正式成为最安全、最具信用的资产。央行可以通过公开市场操作，买卖国债改变货币总量

1.6. 现代中央银行的资产负债表
> **Asset：**
> - 国债（最核心资产，高信用、流动性强，支撑货币发行和货币政策操作）
> - 抵押贷款支持证券（MBS）QE 时购买，降低长期利率、支持住房市场
> - 外汇储备（管理汇率或国际支付流动性）
> - 黄金
>
> **Liabilities：**
> - 流通中的货币（Currency in Circulation）纸币现金，公众持有
> - 银行准备金（Reserves）商业银行在央行的存款，存款乘数形成广义货币
> - 逆回购和其他短期负债
>
> **Equity/Capital：**
> - 央行自有资本、盈余(风险缓冲，用于吸收损失)

这个时候货币体系的基石是社会的共识和信任，他们信任政府未来存在，央行会维持货币稳定和经济稳定

1.7. 由央行资产负债表衍生问题：黄金拿什么买的
央行买黄金用的支付手段主要是是高信用资产，主要包括：

1. **外汇储备**
    - 美元、欧元、日元等国际储备货币
        - 最常用方式，因为黄金国际市场交易多以美元计价

2. **政府国债**
   - 央行可以用自己持有的国债或其他金融资产换取黄金

3. **现有黄金或其他贵金属（少数情况）**
   - 用已有黄金抵押或交换
   - 这种方式一般是央行之间的交易

4. **用负债里的本国货币购买本国黄金**

# 2：关于外汇
2.1. “外汇”（foreign exchange，简称 forex 或 FX）是指以外国货币表示的支付手段和资产，可以用来进行国际间结算、支付或储备

2.2.央行的外汇储备（Foreign Exchange Reserves）主要来自以下几个渠道：
**（1）经常账户盈余（贸易顺差）**
国家出口商品和服务赚取外币，例如美元、欧元、日元等

**（2）资本账户流入（投资和借贷）**

**（3）央行直接干预外汇市场**
- 为了维持汇率稳定，央行在外汇市场买入或卖出本币/外币
- 买入外币 → 外汇储备增加
- 卖出外币 → 外汇储备减少.

**（4）外汇借款或国际机构融资**
- 央行可以向其他国家央行或国际组织借外汇,主要用于流动性管理或应急储备
